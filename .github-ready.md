Since the ONNX model is 29MB, you have options:

1. **Git LFS** (recommended if model won't change often)
   - Install git-lfs
   - Track *.onnx files
   - Users get model automatically

2. **Separate Download** (current approach)
   - Provide download link in README
   - Users download separately
   - Keeps repo lightweight

3. **Releases**
   - Attach model to GitHub release
   - Include download script
